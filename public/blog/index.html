Gun Deaths and Gun Laws: Recursive Partitioning and Regression Tree

By Jenny Listman This project uses a Recursive Partitioning And Regression Tree analysis from the rpart package to examine the effect of 49 gun law subcategories on annual State firearm death rates. Ten subcategories were found to significantly predict State gun death rate decreases or increases.
